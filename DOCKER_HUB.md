# Docker Hub Documentation for Searchboxarr

This document provides all the necessary information for using the Searchboxarr image on Docker Hub.

## Pulling the Image

To pull the Searchboxarr image from Docker Hub, run the following command:
```
docker pull husain3012/searchboxarr
```

## Running the Container

After pulling the image, you can run it using:
```
docker run -d husain3012/searchboxarr
```

## Image Tags

The Searchboxarr image is tagged to allow for specific versions. Use the following command to see available tags:
```
docker pull husain3012/searchboxarr:<tag>
```
Replace `<tag>` with the desired version.

## Documentation

For further documentation, please refer to the [official GitHub repository](https://github.com/husain3012/searchboxarr).
